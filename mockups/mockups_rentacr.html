db_Administrativo
Acceso completo de lectura y escritura en todos los objetos del sistema. Solo para administradores de base de datos.
✔ SELECT en todas las vistas
✔ INSERT, UPDATE, DELETE en tablas
✔ EXECUTE en stored procedures
✔ Gestión de roles y usuarios
✔ Acceso a logs de auditoría
db_Mantenimiento
Permite operaciones CRUD en tablas. Asignado a supervisores y agentes con responsabilidades operativas.
✔ SELECT en tablas y vistas
✔ INSERT, UPDATE, DELETE en tablas
✔ EXECUTE en stored procedures
✗ Gestión de roles y usuarios
✗ Acceso a logs de auditoría
db_LecturaGeneral
Solo lectura sobre vistas. Nunca acceso directo a tablas base. Para agentes con acceso limitado por sucursal (RLS activo).
✔ SELECT en vistas únicamente
✗ SELECT directo en tablas
✗ INSERT, UPDATE, DELETE
✗ EXECUTE en stored procedures
✗ Gestión del sistema
🔐 Dynamic Data Masking activo
Los campos correo electrónico, cédula y dirección están enmascarados para db_LecturaGeneral y db_Mantenimiento. Solo db_Administrativo ve datos completos.
